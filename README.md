# Sneak Peek

## Description
Sneak Peek is a website for fashion enthusiasts and sneakerheads to find their next pair of sneakers to go with their wardrobes.

## Languages Used
* HTML
* CSS
* JavaScript

## Other Technologies Used
* The Sneaker Database API
* The YouTube API
